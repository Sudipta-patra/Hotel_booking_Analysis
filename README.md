# Hotel_booking_Analysis
An end-to-end Hotel Booking Analysis project showcasing data cleaning , how can connect tools , interactive dashboard using Power Bi, Python, PostgreSQL

**📊 Data Analytics Project**

Overview
This project demonstrates a complete end-to-end data analytics workflow. The dataset was imported into Python for data exploration and cleaning, analyzed using SQL, visualized through an interactive Power BI dashboard, and summarized in a professional report and presentation.

The objective of this project is to transform raw data into meaningful insights that support data-driven decision-making.

**Dataset**

Dataset: Hotel Booking

Source: Kaggle

Format: CSV 

Records: 119390

Features: 36

Python- Pandas, Matplotlib, Seaboarn
